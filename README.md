# Pattern-recognition
## はじめに
現在パターン認識について学習している．　  
学習しているだけだとあれなので，実際にコードを書いてみることにした．  
機械学習なら`python`でしょ〜〜っていう世の中なので，今回は`golang`でやってみることにした．　　　　
  
## 今回行うこと．
まずは，ある学習パターンに対して，2クラスに分類するプログラムの実装を行う．

## 環境構築
`golang` の環境構築は割愛．  
`golang`でグラフを描くときには`gonum plot`が便利らしいので，以下でインストール
```
go get gonum.org/v1/plot/...
```
